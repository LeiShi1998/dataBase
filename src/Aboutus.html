About Us

SelenoproteinDB is a selenoprotein database website based on LAMP framework, which belongs to the secondary protein database website.

The site is currently open for data query and data download services.In the query service, keyword query can be used for precise search according to protein name, species, protein family, etc.You can also use the species and protein family table provided by the website to search.Blast pages can also be used for Blast alignment lookups by providing nucleic acid or amino acid sequences.
